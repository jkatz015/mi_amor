## About

This is a simple, love-themed memory match game built with vanilla HTML, CSS, and JavaScript. It's designed to be easy to maintain, code, and deploy - perfect for learning vibe coding together!

## How to Play

1. Click on any card to flip it and reveal an emoji
2. Try to find matching pairs of love-themed emojis
3. Match all 8 pairs to win!
4. Try to complete the game in as few moves as possible

## How to Run Locally

### Option 1: Using Python (Recommended)
```bash
# Navigate to the project folder
cd mi_amor

# Start a simple HTTP server
python3 -m http.server 8000

# Open your browser and go to:
# http://localhost:8000
```

### Option 2: Using Node.js
```bash
# Install a simple server globally
npm install -g serve

# Run the server
serve .

# Open the URL shown in the terminal
```

### Option 3: Just Open the File
Simply double-click on `index.html` to open it directly in your browser!
